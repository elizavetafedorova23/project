Программа имитирует загрузку файла. Во время работы постепенно меняется значение прогресса от 0 до 100 процентов. При каждом изменении вызывается событие ProgressChanged, а подписчик выводит текущее значение в консоль. После завершения загрузки программа выводит сообщение «Готово».

Для запуска нужно открыть проект в Visual Studio или перейти в папку проекта через терминал и выполнить команду dotnet run. После запуска в консоли будет отображаться изменение процента загрузки до 100%.
